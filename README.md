
# react-admin-template 后台管理系统

## react-admin-template 简单上手，可扩展性高

一个可用于生产环境的基于 React19、Ant Design5、Vite6、react-router7、Tailwindcss4，开箱即用的后台管理端项目基础模板

如果对您有一点帮助的，麻烦给下star 🌟 。 万分感谢～～～～

## 技术栈

- React 19
- React-router 7x
- Ant Design 5x
- Tailwindcss 4x
- @reduxjs/toolkit 2x
- Ahooks 3x
- cssinjs、scss
- Axios
- Mock
- Lodash
